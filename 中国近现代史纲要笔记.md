## 导言<a id = "导言"></a>
1. 中国近代史/中国处于半殖民地半封建社会的历史阶段：从1840年鸦片战争爆发到1949年中华人民共和国成立
2. 中国现代史：从1949年中华人民共和国成立以来的历史
3. 中国近现代史的三条主线：
   1. 中国人民为救亡图存和实现中华民族伟大复兴而英勇奋斗、艰辛探索并不断取得伟大成就
   2. 全国各族人民在中国共产党领导下，赢得民族独立，人民解放
   3. 把极度贫困落后的中国逐步改变成持续走向繁荣富强、充满生机活力的社会主义中国
4. 党的最高理想和最终目标：实现共产主义
5. 实现中华民族伟大复兴的必由之路：坚持和发展中国特色社会主义
6. 实现中华民族伟大复兴的根本保证：坚持中国共产党的领导
7. 鸦片战争起中国社会发生的两个根本变化：独立的中国逐渐变成半殖民地的中国，封建的中国逐渐变成半封建的中国
8. 近代中国的主要矛盾：帝国主义和中华民族的矛盾，封建主义和人民大众的矛盾
9. 近代中国最革命的阶级：新兴的工人阶级
10. 中国人民必须完成的两大历史任务：争取民族独立、人民解放，实现国家富强、人民幸福
11. 中国人民和中华民族的最伟大的梦想：实现中华民族伟大复兴的中国梦
12. 压在中国人民头上的三座大山：帝国主义、封建主义、官僚资本主义
13. 新民主主义革命的开端：五四运动
14. 新民主主义革命所要达到的基本目标：建立一个以工人阶级领导为基础的人民民主专政的人民共和国
15. 新民主主义革命时期中国共产党的主要任务：反对帝国主义、封建主义、官僚资本主义，争取民族独立、人民解放，为实现中华民族伟大复兴创造根本条件。
16. 新中国的国体：工人阶级领导的、以工农联盟为基础的人民民主专政的社会主义国家
17. 新中国的政体：人民代表大会制度
18. 社会主义革命和建设时期中国共产党的主要任务：实现从新民主主义到社会主义的转变，进行社会主义革命，推进社会主义建设，为实现中华民族伟大复兴奠定根本政治前提和制度基础
19. 改革开放和社会主义现代化建设新时期的标志：党的十一届三中全会
20. 改革开放和社会主义现代化建设新时期中国共产党的主要任务：继续探索中国建设社会主义的正确道路，解放和发展社会生产力，使人民摆脱贫困，尽快富裕起来，为实现中华民族伟大复兴提供充满新的活力的体制保证和快速发展的物质条件
21. 决定当代中国前途和命运的关键一招：改革开放
22. 中国特色社会主义进入新时代的标志：党的十八大
23. 中国特色社会主义新时代党的主要任务：实现第一个百年奋斗目标，开启实现第二个百年奋斗目标，朝着实现中华民族伟大复兴的宏伟目标继续前进

## 第一章：进入近代后中华民族的磨难与抗争<a id = "第一章"></a>
### 第一节：鸦片战争前后的中国和世界
1. 封建社会
```
经济主导地位：封建地主所有制经济
基本生产结构：以个体家庭为单位并与家庭手工业牢固结合的小农经济
经济主要地位：自给自足的自然经济
政治的基本特征：实行高度中央集权的封建君主专制制度
社会结构特点：族权和政权相结合的封建宗法等级制度，其核心是宗族家长制，突出君权、父权、夫权
```
2. 鸦片战争后签订的条约
```
中英《南京条约》
中英《虎门条约》

中法《黄埔条约》
```

2. 军事侵略方式

   3. 扶植、收买代理人

   1. 控制中国的通商口岸
   2. 剥夺中国关税自主权
   3. 实行商品倾销和资本输出
   4. 操纵中国的经济命脉
8. 文化渗透方式
   1. 披着宗教外衣进行侵略活动
   2. 为侵略中国制造舆论

   2. 中华民族进行的不屈不挠的反侵略斗争（根本原因）

## 第二章：不同社会力量对国家出路的早期探索<a id = "第二章"></a>
### 第一节：太平天国运动的起落
1. 过程
```
1851年1月：金田起义，建号太平天国
1853年3月：占领南京，定为首都，改名天京，正式宣告太平天国成立
1856年9月：天京事变，严重削弱了太平天国的领导和军事力量，成为太平天国由盛转衰的分水岭
1864年7月：天京陷落，太平天国起义失败
```
2. 性质：反对清政府腐朽统治和地主阶级压迫剥削的正义战争
3. 两个文件：《天朝田亩制度》和《资政新篇》
4. 历史意义：
   1. 沉重打击了封建统治阶级，强烈撼动了清王朝的统治根基
   2. 是中国旧式农民战争的最高峰
   3. 冲击了孔子和儒家经典的正统权威，一定程度上削弱了封建统治的精神支柱
   4. 有力的打击了外国侵略势力
5. 失败教训：在半殖民地半封建社会的中国，农民具有伟大的革命潜力，但它自身不能担负起领导反帝反封建斗争取得胜利的重任，单纯的农民战争不可能完成争取民族独立和人民解放的历史人物
### 第二节：洋务运动的兴衰
1. 过程
```
代表任务：奕訢、曾国藩、李鸿章、左宗棠、张之洞等
指导思想：中学为体，西学为用
措施：1. 兴办近代企业 2.建立新式陆海军 3.创办新式学堂，派遣留学生
```
2. 洋务运动兴办的近代企业
   1. 军工企业：安庆军械所，江南制造总局，金陵机器局，福州船政局（附设有船政学堂），天津机器局，湖北枪炮厂
   2. 民用企业：轮船招商局，开平矿务局，天津电报局，上海机器织布局
3. 洋务运动的作用：
   1. 客观上对中国的早期工业和民族资本主义的发展起到了某些促进作用
   2. 是中国近代教育的开始，给当时的中国带来了新的知识，使人们开拓了眼界
   3. 传统的“重本抑末”等观念受到冲击，社会风气和价值观念发生变化，工商者的地位上升，有利于资本主义经济的发展，也有利于社会风气的改变
4. 洋务运动失败的原因：
   1. 洋务运动具有封建性
   2. 洋务运动对列强具有依赖性
   3. 洋务企业的管理具有腐朽性
### 第三节：维新运动的兴起与夭折
1. 资产阶级维新派兴起的时代背景：
   1. 新兴的民族资产阶级迫切要求挣脱外国资本主义和国内封建主义的压迫和束缚，为中国发展资本主义开辟道路
   2. 在内忧外患的冲击和甲午战争的惨败中人民逐步形成了一个共识：要救国，只有维新，要维新，只有学外国
2. 过程
```
代表人物：康有为、梁启超、谭嗣同、严复
行动：1. 向皇帝上书 2. 著书立说 3. 介绍外国变法的经验教训 4. 办学会 5. 设学堂 6. 办报纸
```
3. 维新派主张：
   1. 维新变法，革除积弊，才能挽救中国所面临的危亡局面，以图求存和自强
   2. 主张兴绅权，即首先要为正在向资产阶级转化的士绅争取政治地位，认为只有君主立宪制度才是当时中国理想的政治方案，要兴民权、设议院，实行君主立宪
   3. 废八股、改科举，办学堂、兴西学
4. 戊戌六君子：谭嗣同、刘光第、林旭、杨锐、杨深秀、康广仁
5. 维新运动的意义：
   1. 戊戌维新运动是一次爱国救亡运动
   2. 戊戌维新运动是一次资产阶级性质的政治改良运动
   3. 戊戌维新运动是一次思想启蒙运动
6. 失败原因：维新派自身的局限以及慈禧太后为首的强大的守旧势力的反对
7. 维新派本身的局限性：
   1. 不敢否定封建主义
   2. 对帝国主义抱有幻想
   3. 惧怕人民群众
8. 维新运动失败的教训：在半殖民地半封建社会的旧中国，企图通过统治者走自上而下的改良道路，是根本行不通的，必须用革命的手段，推翻帝国主义、封建主义联合统治的半殖民地半封建的社会制度

## 第三章：辛亥革命与君主专制制度的终结<a id = "第三章"></a>
### 第一节：举起近代民族民主革命的旗帜
1. 辛亥革命爆发的历史条件：
   1. 民族危机加深、社会矛盾激化
   2. 清末新政的破产
   3. 资产阶级革命派的阶级基础和骨干力量
2. 清末新政破产的主要原因：清政府改革的根本目的是延续其反动统治
3. 资产阶级革命团体
```
1894年11月：孙中山在檀香山创立兴中会
其他革命团体：华兴会、科学补习所、光复会、岳王会
1905年8月20日：孙中山、黄兴、宋教仁在日本东京成立中国同盟会，是近代中国第一个领导资产阶级革命的全国性政党
```
4. 宣传革命的书籍报刊
```
章炳麟《驳康有为论革命书》
邹容《革命军》
陈天华《猛回头》《警世钟》
```
5. 同盟会的政治纲领：驱除鞑虏，恢复中华，创立民国，平均地权
6. 三民主义：民族主义，民权主义，民生主义
### 第二节：辛亥革命与中华民国的建立
1. 过程
```
1911年4月27日：广州起义
1911年5月：保路运动，以四川最为激烈
1911年10月10日：武昌起义
1912年2月12日：清帝被迫退位
```
2. 资产阶级革命派的特点：既表现出了革命性和勇敢精神，又暴露出了软弱性和妥协态度
3. 辛亥革命的性质：辛亥革命是资产阶级领导的以反对封建君主专制制度、建立资产阶级共和国为目的的革命，是一次比较完全意义上的资产阶级民主革命
4. 辛亥革命的历史意义：
   1. 辛亥革命推翻了封建势力的政治代表、帝国主义在中国的代理人清王朝的统治，沉重打击了中外反动势力，是中国反动统治者在政治上乱了阵脚
   2. 辛亥革命结束了中国延续两千多年的封建君主专制制度，建立了中国历史上第一个资产阶级共和政府，使民主共和的观念深入人心
   3. 辛亥革命推动了中国人民的思想解放
   4. 辛亥革命推动了中国的社会变革，促进中国的社会经济、思想习惯和社会风俗等方面发生了新的积极变化
   5. 辛亥革命不仅在一定程度上打击了帝国主义的侵略势力，而且推动了亚洲各国解放运动的高涨
5. 辛亥革命失败的原因：
   1. 根本上，在帝国主义时代，在半殖民地半封建社会的中国，资本主义的建国方案时行不通的
   2. 没有彻底的提出反帝反封建的革命纲领
   3. 不能充分的发动和依靠人民群众
   4. 不能建立坚强的革命政党
6. 辛亥革命的历史教训：资产阶级共和国的方案没有能够救中国，先进的中国人需要进行新的探索，为中国谋求新的出路
### 第三节：北洋军阀统治与旧民主主义革命的失败
1. 过程
```
1912年4月1日：孙中山辞职，随后临时政府迁往北京
1912年8月：宋教仁改组形成国民党
1913年3月：宋教仁遇刺
1913年7月至9月：袁世凯镇压二次革命
1914年：孙中山在日本组织中华革命党
1915年5月：袁世凯接受“二十一条”
1915年12月25日：蔡锷组织护国军，护国运动爆发
1916年1月1日：袁世凯举行登基大典，复辟帝制
1916年3月：袁世凯败亡
1917年6月：张勋复辟
1917年9月：孙中山在广州成立护法军政府，护法运动爆发
1919年10月：孙中山将中华革命党改组为中国国民党
```
1. 军阀割据局面形成的原因
   1. 中国主要是地方性的农业经济而没有形成统一的资本主义市场
   2. 帝国主义国家在中国采取划分势力范围的分裂剥削政策

## 第四章：中国共产党成立和中国革命新局面<a id = "第四章"></a>
### 第一节：新文化运动和五四运动
1. 新文化运动
```
开始标志：陈独秀在上海创办《青年杂志》
主要阵地：《新青年杂志》和北京大学
口号：民主与科学
``` 
2. 新文化运动的局限：
   1. 新文化运动的倡导者批判孔学是为了给中国发展资本主义扫清障碍
   2. 新文化运动的倡导者把改造国民性置于优先地位
   3. 许多领导者没有马克思主义的批判精神，仍旧使用资本主义的方法
3. 马克思主义在中国的传播
```
率先举起马克思主义旗帜：李大钊
著作：《我的马克思主义观》
```
4. 五四运动爆发的条件
   1. 新的社会力量的壮大
   2. 新文化运动掀起的思想解放的潮流
   3. 俄国十月革命对中国的影响
5. 五四运动导火线：巴黎和会上中国外交的失败
